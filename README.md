# Geometric Art Generator

Geometric Art Generator built with HTML, CSS and JavaScript.

![Geometric Art Generator Screenshot](./Screenshot%202024-04-27%20154451.png)

> A small, browser-based generator that creates colorful geometric patterns using HTML, CSS and vanilla JavaScript. Great for backgrounds, generative art experiments and learning DOM-driven drawing techniques.

## Features

- Generates abstract geometric art using simple HTML/CSS/JS.
- Lightweight and dependency-free — just open the files in your browser.
- Easily customizable: tweak colors, shapes, layout and animation from the JavaScript and CSS.

## Demo / Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/BinaryVortex/Geometric-Art-Generator.git
   cd Geometric-Art-Generator
   ```

2. Open the project in your browser:

   - Double-click `index.html`, or
   - Serve it with a simple static server (recommended for some browsers):

     ```bash
     # Python 3
     python -m http.server 8000
     # then open http://localhost:8000
     ```

3. Edit the JavaScript/CSS files to change colors, shapes, sizes and animation timing. The main logic lives in `script.js` (or similarly named file) and the visual styles in `styles.css` — look for comments in those files.

## Customization tips

- Colors: change CSS variables or the color arrays in the JavaScript to generate different palettes.
- Shape types: add or modify the functions that create shapes to introduce triangles, polygons, circles, etc.
- Performance: reduce the number of elements or disable heavy animations for slower machines.
